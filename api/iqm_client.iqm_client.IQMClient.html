iqm_client.iqm_client.IQMClient¶

class iqm_client.iqm_client.IQMClient(url, settings)¶

Bases: object

Provides access to IQM quantum computers.

Parameters

  • url (str) – Endpoint for accessing the server. Has to start with http or https.

  • settings (dict[str, Any]) – Settings for the quantum computer, in IQM JSON format.

submit_circuit(circuit, qubit_mapping, shots=1)¶

Submits a quantum circuit to be executed on a quantum computer.

Parameters

  • circuit (iqm_client.iqm_client.Circuit) – circuit to be executed

  • qubit_mapping (list) – mapping of human-readable qubit names in circuit to physical qubit names

  • shots (int) – number of times circuit is executed

Returns

ID for the created task. This ID is needed to query the status and the execution results.

Return type

uuid.UUID
